# ⚡ FieldLab: Sports Performance & Injury Prevention

**FieldLab** es una plataforma avanzada de gestión de carga y análisis biomecánico diseñada para el alto rendimiento (Rugby, Hockey, Fútbol). Combina visión artificial y ciencia del deporte para optimizar el rendimiento y reducir el riesgo de lesiones.

## 🚀 Características Principales
* **Control de Carga (ACWR):** Monitoreo del ratio carga aguda/crónica para evitar el sobreentrenamiento.
* **Análisis de Asimetría Dinámica:** Detección de desbalances en saltos (Bosco) y aceleración (Test de 3 pasos).
* **Visión Artificial (MediaPipe):** Evaluación goniométrica y cinemática en tiempo real sin sensores externos.
* **VBT (Velocity Based Training):** Seguimiento de la velocidad de ejecución para el entrenamiento de potencia.

## 🧠 Arquitectura del Agente (AI-Ready)
Este repositorio está optimizado para ser operado por asistentes de IA (como Claude). 
Toda la lógica científica y los estándares visuales se encuentran en la carpeta `/agent`:
* `/agent/instructions.md`: Identidad y rol del desarrollador.
* `/agent/ui-standards.json`: Guía de estilos y diseño técnico.
* `/agent/skills/biomechanics-logic.md`: Umbrales, fórmulas y protocolos validados.

## 🛠️ Stack Tecnológico
* **Frontend:** React + Tailwind CSS.
* **Iconografía:** Lucide-React.
* **Gráficos:** Recharts.
* **Procesamiento:** MediaPipe + TensorFlow.js.
